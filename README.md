# cs348project

# Setup/Run Instructions

## Backend

First time:

```
run startup.sql in mysql
cd server
npm install
```

Create a `.env` file in the `server` directory with the following contents:
`DB_PASSWORD=your_password`

Run:
`node ./index.js`

## Frontend/Client

First time:

```
cd client
npm install
```

Run:
`npm run dev`
